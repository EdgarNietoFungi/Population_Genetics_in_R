Preface

We had a great time writing this primer thanks to the ready availability of many open source tools. This primer was written using open source tools including Markdown, Rstudio IDE, Knitr, the R statistical computing and graphing language, knitcitations, poppr and other dependent R packages.

If you find errors or lacking information please let us know. We are actively working to improve this primer and poppr. We are also delighted to hear about features in poppr you might want to see, although we cannot promise that we can implement these. At the same time we have already implemented several user suggested features that much improved poppr. We hope you find this primer useful in your own work!

Niklaus J. Grünwald, Zhian N. Kamvar, and Sydney E. Everhart
